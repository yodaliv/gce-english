# Ace it!

*Ace it!* is a quizz app to help you prepare for the AEIS examination through targeted practice. Unlike other quizz app, *Ace it!* automatically generates quizzes adapted to the learnersâ€™ level and objectives.

## Technical Stack

We are using [Ionic framework v5](https://ionicframework.com) to build this hybrid mobile application to publish Android and iOS version. We are all based on [Angular v9](https://angular.io).
